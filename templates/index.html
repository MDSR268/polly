Advanced Poll Creation

Named polls with up to 10 custom options, full-size image uploads, custom emojis, and rich Discord embeds with real-time vote counts.

Smart Scheduling System

Calendar and time picker for precise scheduling, automatic poll opening/closing, timezone-aware system with background job processing.

Anonymous Polls

Hide results until poll ends, protect voter privacy while maintaining poll integrity and comprehensive analytics for administrators.

Multiple Choice Support

Allow users to select multiple options, perfect for preference surveys, availability polls, and complex decision-making scenarios.

Role Ping System

Automatically ping specific roles when polls open or close, ensuring maximum participation and timely notifications for important decisions.

Full-Size Images

Upload and display full-size images (up to 8MB) with custom message text, not tiny thumbnails. Perfect for visual polls and comparisons.

Real-Time Dashboard

Beautiful Bootstrap interface with HTMX for live updates without page refreshes. View poll history, results, and analytics in real-time.

Advanced Analytics

Comprehensive vote tracking, user participation metrics, spreadsheet-style results view, and detailed CSV/JSON export capabilities.

User Preferences

Remembers last used server, channel, timezone, and role settings. Smart defaults and priority timezone system for seamless poll creation.

JSON Import/Export

Import polls from JSON files with comprehensive validation, export existing polls for backup or templating. Perfect for bulk operations.

Poll Templates

Create templates from existing polls for quick reuse. Perfect for recurring polls like weekly meetings, team lunches, or regular surveys.

Mobile Responsive

Fully responsive design that works perfectly on desktop, tablet, and mobile devices. Manage polls from anywhere with full functionality.

Redis Caching
